# CliForTodoist
## Purpose
This project deals with bring a interface to todoist.com that also works ofline, utilizeing the officially suported API.
## Basics
```bash
$ todo [show]
$ todo add "Task name" [-p project#] [-d dateString]
$ todo done task#
$ todo sync
```

## Install
```bash
$ bash install.sh
```

## Uninstall
```bash
$ bash uninstall.sh
```
